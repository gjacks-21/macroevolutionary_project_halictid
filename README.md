# macroevolutionary_project_halictid
Welcome! Here, I showcase my macroevolutionary investigation into halictid bees test whether eusocial lineages of halictid bees exhibit higher diversification rates than their non-eusocial relatives, and if the trait of eusociality has an impact on speciation, extinction, and/or character transition rates.
I used R to construct a time-calibrated phylogenetic tree, performed diversification rate analysis using MEDUSA, and zoomed in on trait-dependent diversification specifically using BiSSE.
